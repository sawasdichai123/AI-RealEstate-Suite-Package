Introduction

AI Real Estate Marketing Suite is a powerful Next.js application designed for real estate agents to generate high-quality marketing content using OpenAI's GPT-4o technology.

Prerequisites

  • Node.js: v18.x or higher
  • Package Manager: npm or yarn
  • OpenAI API Key: Required for AI generation

Installation Guide

Follow these steps to get the project running on your local machine:

Step 1: Extract the Source_Code.zip file
Step 2: Open the folder in VS Code
Step 3: Run: npm install
Step 4: Rename .env.example to .env.local
Step 5: Add your OpenAI API Key to the .env.local file
Step 6: Run: npm run dev

Configuration (.env)

Edit your .env.local file with the following variables:

OPENAI_API_KEY=your_api_key_here
NEXT_PUBLIC_APP_URL=http://localhost:3000
Pro Tip: Never commit your .env.local file to version control. Keep your API keys secure!

Deployment (Vercel)

This project is optimized for Vercel. Simply connect your GitHub repository to Vercel, add your Environment Variables, and click "Deploy".
